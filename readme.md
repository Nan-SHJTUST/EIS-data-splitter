# 🧪 EIS Data Splitter (Web App)

这是一个专门为电化学阻抗谱（EIS）设计的在线数据处理工具。它可以自动识别多轮扫描（Sweep）数据，支持多文件上传，并能通过网页交互快速提取频率、阻抗实部和虚部。

## ✨ 功能特点
- **多文件上传**：支持批量处理多个工作站导出的 CSV 文件。
- **动态列映射**：无需修改代码，直接在网页上指定数据所在的行和列。
- **智能轮次识别**：根据频率变化自动拆分多轮循环实验数据。
- **实时预览**：自动生成符合电化学惯例的 Nyquist 图（-Z'' vs Z'）。
- **打包下载**：一键导出所有处理后的数据为标准的 `.txt` 格式压缩包。
